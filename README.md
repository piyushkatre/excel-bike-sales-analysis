# 🚲 Bike Sales Insights Dashboard

## 📌 Project Overview
This project analyzes customer bike purchasing behavior using Excel dashboards, Pivot Tables, Pivot Charts, slicers, and KPI cards.

The dashboard helps identify patterns related to:
- Age groups
- Income levels
- Commute distance
- Education
- Marital status

---

## 📊 Dashboard Features
- Interactive slicers
- KPI cards
- Pivot Charts
- Customer segmentation analysis
- Business insights extraction

---

## 📈 Key Insights
✓ Middle-aged customers purchase bikes most frequently.

✓ Higher-income customers show greater purchase tendency.

✓ Short commute distances correlate with higher bike purchases.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
